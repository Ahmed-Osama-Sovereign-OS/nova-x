# NOVA X

Quantum Reality Engine v0.1

---

## Overview

NOVA X is an experimental quantum-inspired visual engine.

The project simulates:
- probability fields
- quantum waves
- reality distortion
- energy systems
- cinematic particle universes

inside a real-time interactive 3D environment.

---

## Features

- Real-time 3D universe
- Quantum particle system
- Energy rings
- Reality distortion effects
- Cinematic boot sequence
- Interactive mouse reactions
- Probability burst system
- Dynamic AI behavior
- Quantum control panel
- Lightweight rendering engine

---

| Control | Effect |
